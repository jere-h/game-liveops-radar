        Expected CSV format

        Calendar CSV

        One row per initiative. Dates are ISO YYYY-MM-DD; the window is inclusive of both ends.

        initiative_id,name,start_date,end_date,segment_predicate
summer_sale,Summer Sale,2026-07-01,2026-07-14,
double_xp,Double XP Weekend,2026-07-10,2026-07-12,
        • initiative_id — unique id (string)
        • name — human-readable label
        • start_date, end_date — ISO dates, inclusive
        • segment_predicate — optional, may be blank

        Assignments CSV

        One row per player enrolled in an initiative. Duplicate (player, initiative) rows are de-duplicated.

        initiative_id,player_id
summer_sale,player_001
double_xp,player_001
        • initiative_id — must match a calendar initiative
        • player_id — enrolled player id (string)
